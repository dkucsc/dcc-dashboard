Help

You can do a bulk download from the Redwood Client using the manifest. To get the manifest, go to the File Browser, scroll to the bottom, and click "Download Manifest".

Setting up the Redwood Client

You can download the Redwood Client using docker. To download the client, use from the command line:
docker pull benjaminran/redwood-client

Getting the token

Contact the admin to get an access token if you don't have one.

Perform Bulk Download

To do a bulk download, run the following docker command:
docker run --rm -it -e ACCESS_TOKEN=<my_token> -e REDWOOD_ENDPOINT=storage.ucsc-cgl.org -v `pwd`/<my/data/path>:/dcc/data benjaminran/redwood-client bash bulk_download <my_manifest> data/
This will download the data in the folder specified in the current directory.
